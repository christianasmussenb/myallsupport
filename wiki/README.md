## 📖 Cómo Usar Este Wiki

### En GitHub

Si este proyecto está en GitHub, puedes ver el wiki en:
- El directorio `/wiki` del repositorio
- O en la sección Wiki de GitHub (si está habilitada)

### Localmente

```bash
# Clonar el repositorio
git clone https://github.com/christianasmussenb/myallsupport.git
cd myallsupport/wiki

# Ver cualquier archivo .md con tu editor favorito
cat Home.md
# o abrirlo en VS Code, Sublime, etc.
```

### En un Navegador

Puedes usar extensiones de Chrome/Firefox para renderizar Markdown:
- **Markdown Viewer** (Chrome)
- **Markdown Viewer** (Firefox)

O usar herramientas online:
- https://dillinger.io/
- https://stackedit.io/

## 🔄 Mantener el Wiki Actualizado

- Se agregan nuevas funcionalidades al sitio

## 📝 Formato del Wiki

Todos los archivos usan **Markdown** con las siguientes convenciones:

### Encabezados
```markdown
# Título Principal (H1)
## Sección (H2)
### Subsección (H3)
```

### Listas
```markdown
- Elemento de lista
  - Sub-elemento
  
1. Lista numerada
2. Segundo item
```

### Código
```markdown
`código inline`

​```javascript
// Bloque de código
const variable = 'valor';
​```
```

### Enlaces
```markdown
[Texto del enlace](URL)
[Enlace interno](Archivo.md)
[Enlace a sección](#seccion)
```

### Imágenes
```markdown
![Alt text](ruta/a/imagen.png)
```

### Emojis
```markdown
🚀 📚 ✅ ❌ 💡 🔧 
```
