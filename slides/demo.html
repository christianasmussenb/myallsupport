import Anthropic from '@anthropic-ai/sdk'

const client = new Anthropic()

const agent = await client.beta.agents.create({
  model:   "claude-sonnet-4-6",
  tools:   [webSearch, codeExec, fileWrite],
  memory:  { type: "persistent", ttl: "7d" },
})

const run = await agent.run("Research and write a report on AGI timelines")
console.log(run.output)

What Makes an Agent Trustworthy

Three pillars that distinguish safe, reliable AI from unreliable systems

🔍
Transparency
Shows its reasoning, cites sources, and explains every action it takes before taking it.
🛡
Corrigibility
Stops immediately when asked, defers to human judgment, and never pursues goals beyond its mandate.
📊
Calibration
Knows what it doesn't know. Expresses uncertainty accurately rather than confabulating answers.
🤖
